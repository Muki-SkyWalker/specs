class MockitoMocker
[source: org/mockito/MockitoMocker.scala]
class MockitoMocker
extends AnyRef
This class is created to get an access to the MOCKING_PROGRESS Mockito package variable which is package protected. Then it delegates all the methods to the Mockito static methods.
