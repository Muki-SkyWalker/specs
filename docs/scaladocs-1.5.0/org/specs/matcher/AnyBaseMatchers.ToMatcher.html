class ToMatcher[S, T](f : (S) => Matcher[T])
extends AnyRef
The ToMatcher class allows to combine functions returning matchers, or a function returning a matcher and a matcher.

For example:((beEqualTo(_:Int)) or (be_>(_:Int)))(3)
