trait CalledMatchers
extends CalledInOrderMatchers
This trait provides methods to declare expectations on mock calls: mockedList.get(0) was called mockedList.get(0) wasnt called mockedList.get(0) was notCalled where called is a Matcher which accepts supplementary methods to change the Mockito verification method: mockedList.get(0) was called.once // similar to was called mockedList.get(0) was called.twice mockedList.get(0) was called(3.times) mockedList.get(0) was called.atLeastOnce // or called.atLeast(oncee) mockedList.get(0) was called.atLeastTwice mockedList.get(0) was called.atLeast(3.times) mockedList.get(0) was called.atMostOnce mockedList.get(0) was called.atMostTwice mockedList.get(0) was called.atMost(3.times) mockedList.get(0) was called.exclusively // equivalent to called(0.timess)
