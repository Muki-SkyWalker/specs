trait MockitoLifeCycle
extends ExampleLifeCycle
This trait allows the initialization of mocks when defined with an annotation: @Mock val l: List[String] = null The beforeTest method is overriden instead of the beforeExample method. This doesn't make a big difference but it helps the compiler in the Eclipse plugin not to crash too often.
