specs 1.5.0 API

This document is the API specification for specs 1.5.0 API

Class Summary
trait JavaConversions extends AnyRef
This trait provides some implicit conversions between Java collections and scala.collection.List
Object Summary
object ExtendedIterable extends AnyRef
The ExtendedIterable object offers utility methods applicable to iterables like:
  • toStream
  • toDeepString: calls toString recursively on the iterable elements
  • sameElementsAs: compares 2 iterables recursively
object ExtendedList extends AnyRef
This object provides useful functions for Lists, like:
  • everyOrder: returns all permutations of the list elements
  • mix: from one list and one element, returns all the lists created by inserting this element in the original list
  • removeFirst: returns a list minus the first element matching a given element according to a predicate
object JavaCollectionsConversion extends JavaConversions
