def addExpectation : Example
def addExpectation [T](expectable : scala.Option[Expectable[T]]) : Example
Add an expectation to the last created example. If there is none, create a new one with the forExample function. If an expectable is given, then this expectable should be assigned the created example. This is the case when adding an expectation for a one-liner jmock expectation:
     // in the expects method a new expectation is created with isExpectation
     // see org.specs.mock.JMocker
     classOf[MyClass].expects(one(_).method) in { _.method }
