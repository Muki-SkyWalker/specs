Expectations

trait Expectations extends Matchers with OrResults with ExpectableFactory with DetailedFailures

This trait can be used to access Matchers functionalities outside a Specification. For example like this:

trait Functions extends Expectations { def bar(name: String) = name.length < 4 mustBe true } object Foo extends Specification with Functions { bar("Foo") bar("FooFoo") }
