class ExampleTestCase(val example : Examples, description : java.lang.String)
extends junit.framework.TestCase with AnyRef
A ExampleTestCase reports the result of an example

It overrides the run method from junit.framework.TestCase to add errors and failures to a junit.framework.TestResult object When possible, the description of the subexamples are added to their failures and skipped exceptions
