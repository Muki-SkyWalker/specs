trait ScalaTest
extends SpecsFilter with FailOrSkip with org.scalatest.Suite
This trait is a ScalaTest suite which is build from one or more specifications (provided by the inherited SpecsHolder) The subspecifications, as well the system under test (sus) are provided as nested suites Usage: class mySpecRunner extends Runner(mySpec) with ScalaTest
