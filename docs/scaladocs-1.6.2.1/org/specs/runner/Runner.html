class Runner

class Runner(val specifications : scala.Seq[Specification], val reporters : scala.Seq[Reporter])
extends Reporter
The Runner class is an abstract class referencing one or several specifications to run. It should be extended with at least one runner trait which will use the specifications. Usage: class mySpecRunner extends Runner(mySpec) with JUnit with Console with ScalaTest or class mySpecRunner extends Runner(mySpec) with JUnit with Xml with ScalaTest which will also output the results in an xml file Then mySpecRunner class can be executed in many ways:
  • scala -cp ... -e "new mySpecRunner.reportSpecs"
  • java -cp ... org.scalatest.Runner -g -s mySpecRunner
  • java -cp ... org.junit.runner.JUnitCore mySpecRunner
It is annotated with a JUnit annotation because JUnit requires that annotation should be placed on the class which will be executed. In the example above, Runner(mySpec) is the only class; JUnit, Console and ScalaTest are all traits.
