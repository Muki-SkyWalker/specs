trait MapMatchers
extends MapBaseMatchers with MapBeHaveMatchers
The MapMatchers trait provides matchers which are applicable to Map objects
It currently accepts any Iterable[(K, V)] whereas it should only accept Map[K, V]. This is because the implicit defs in the BaseSpecification trait are picking up iterables in general
