This document is the API specification for specs 1.6.2.1 API

Class Summary
trait MockFileSystem extends FileSystem
The MockFileSystem trait mocks the FileSystem by storing a Map[path, content] representing the content of the FileSystem
trait MockOutput extends Output
The MockOutput trait catches all messages printed with the Output trait
trait MockWriter extends java.io.Writer with AnyRef
The MockWriter writes all the content written with the Writer interface to the messages: Queue[String] attribute.
