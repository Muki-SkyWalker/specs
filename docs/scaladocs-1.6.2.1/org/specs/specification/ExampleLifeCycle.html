trait ExampleLifeCycle
extends LifeCycle with ExampleStructure
Implementation of a LifeCycle backed up by an ExampleStructure which can be added any subexample created during the Example execution. It will also check that the executed example contains expectations, otherwise will mark the example as "PENDING"
