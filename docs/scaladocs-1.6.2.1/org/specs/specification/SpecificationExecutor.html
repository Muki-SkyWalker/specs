trait SpecificationExecutor
extends LifeCycle
This trait executes an example by cloning the enclosing specification first. This way, the example is executed in a total isolation so as not to share local variables between examples and avoid side-effects. Warning: this works by considering that the "examples" method is stable on a BaseSpecification and will always return the same examples in the same order
