specs 1.6.2.1 API

This document is the API specification for specs 1.6.2.1 API

Class Summary
trait ConsoleOutput extends Output
Implementation of the Output trait using the Console object
trait FileReader extends AnyRef
The FileReader trait provides functions to read files
trait FileSystem extends FileReader with FileWriter
The FileSystem trait abstracts file system operations to allow easier mocking of file system related functionalities.
trait FileWriter extends AnyRef
The FileWriter trait provides functions to write files
trait Output extends AnyRef
Abstract trait representing an output with standard print functions
Object Summary
object fs extends FileSystem
The fs object offers a simple interface to the file system (see the description of the FileSystem trait)
