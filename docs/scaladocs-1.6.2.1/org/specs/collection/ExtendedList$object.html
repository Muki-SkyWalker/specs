object ExtendedList
extends AnyRef
This object provides useful functions for Lists, like:
  • everyOrder: return all permutations of the list elements
  • mix: from one list and one element, return all the lists created by inserting this element in the original list
  • removeFirst: return a list minus the first element matching a given element according to a predicate

def optimum[T](list : scala.List[T], f : (T) => Int, compare : (Int, Int) => Boolean, currentOptimum : Int, optimumElement : scala.Option[T]) : (scala.Option[T], Int)
Find the optimum element of a list according to a valuation function and a comparison function to determine if we require the least or best values.
Parameters
list - list of elements
f - optimization function returning a value for each element of the list
compare - comparison function indicating how values should be compared to get the optimum. For example (_>_) will get the maximum
currentOptimum - current optimum value. This can be 0 when getting the maximum
optimumElement - current optimum element if found. This is None when starting the computation
Returns
the optimum element and value
